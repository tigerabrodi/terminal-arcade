# terminal-arcade

Retro arcade games in your terminal. Split your terminal, play while your AI works.

## Install & Play

```
bunx terminal-arcade
```

## Games

- **Dino Run** - Jump over obstacles in the desert
- **Flappy Bird** - Flap through the pipes
- **Snake** - Eat food and grow without hitting yourself
- **Pong** - Classic paddle ball against the CPU
- **Tetris** - Stack and clear falling blocks

## Controls

- **Arrow keys** / **WASD** - Move
- **Space** - Jump / Flap / Hard drop
- **R** - Restart after game over
- **ESC** - Back to menu (quit from menu)

## Built With

- [Bun](https://bun.sh) - Runtime
- [OpenTUI](https://opentui.com) - Terminal UI framework
- TypeScript
